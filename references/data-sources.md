# 数据源架构

## 三级自动容灾

| 优先级 | 数据源 | 底层接口 | 需要 Token | 说明 |
| --- | --- | --- | :---: | --- |
| 0（最优先） | **妙想金融** | 东财妙想 API | ✅ `MX_APIKEY` | 覆盖行情 + 财务数据 + 主力资金 + 估值 |
| 1（备选） | Efinance | 东方财富 | ❌ | 覆盖日K、实时行情、板块排名、市场统计 |
| 2（备选） | AkShare | 东方财富 + 新浪 | ❌ | 额外支持筹码分布，东财+新浪双通道 |

> - 配置 `MX_APIKEY` 后妙想自动成为最高优先级；未配置时从 Efinance 开始
> - 任意数据源异常时自动切换到下一级，无需人工干预

## 妙想金融数据源

- **API 端点**：`https://mkapi2.dfcfs.com/finskillshub/api/claw/query`
- **查询方式**：自然语言（POST JSON `{"toolQuery": "查询语句"}`）
- **认证方式**：HTTP Header `apikey`
- **特点**：无需指定接口编码，直接用中文描述数据需求，API 自动识别证券和指标
- **限制**：每日调用次数有上限，不支持筹码分布、板块排名、市场统计

## 反封禁策略

- 随机 User-Agent 轮换（5 组 UA）
- 请求间随机 Sleep（1.5 ~ 3.0 秒抖动）
- 全市场实时行情缓存（TTL 600 秒）
- 涨停/跌停精确计算（科创板/创业板 20%、北交所 30%、ST 5%、普通 10%）

## 网络环境适配

- 妙想 API（mkapi2.dfcfs.com）需要能访问东财服务器
- 东方财富接口（push2.eastmoney.com）在企业内网/代理环境下可能受限
- 新浪接口（vip.stock.finance.sina.com.cn）通常不受影响
- 系统会自动回退至可用通道，日志输出 `[数据源切换]` 提示
